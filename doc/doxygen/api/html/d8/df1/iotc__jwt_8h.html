Create a JWT.

Creates a JWT string. To connect to Cloud IoT Core. Pass the JWT to iotc_connect as the password parameter.

This function invokes the BSP implementations of iotc_bsp_sha256(), iotc_bsp_ecc() and iotc_bsp_base64_encode_urlsafe() to enable string encoding and signatures.

This function only supports ES256 key types.

Parameters
[in]expiration_period_secThe number of seconds before this JWT expires.
[in]project_idThe GCP project ID.
[in]private_key_dataES256 <a href"https://cloud.google.com/iot/docs/how-tos/credentials/keys">private key data.
[in,out]dst_jwt_bufA pointer to a buffer that stores a formatted and signed JWT.
[in]dst_jwt_buf_lenThe length of the dst_jwt_buf buffer, in bytes.
[out]bytes_writtenThe number of bytes written to dst_jwt_buf.
Return values
IOTC_STATE_OKA JWT is successfully generated.
IOTC_INVALID_PARAMETERThe project_id, private_key_data or dst_jwt_buf parameters are NULL, or a crypto BSP function returns IOTC_BSP_CRYPTO_INVALID_INPUT_PARAMETER_ERROR.
IOTC_ALG_NOT_SUPPORTED_ERRORThe provided private key isn't an ES256 key.
IOTC_NULL_KEY_DATA_ERRORThe provided private key is a PEM file but the crypto_key_union pointer is NULL.
IOTC_NOT_IMPLEMENTEDThe crypto_key_union pointer type is unknown.
IOTC_BUFFER_TOO_SMALL_ERRORThe provided buffer is too small for the JWT.
