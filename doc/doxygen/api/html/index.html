Google Cloud IoT Device SDK for Embedded C API

Welcome

The Device SDK API securely connects client applications to Cloud IoT Core in order to publish and subscribe to messages via MQTT.

To port the Device SDK to your device, review the Board Support Package (BSP) reference and porting guide.

Getting started

All of the standard Device SDK API functions are in the /include directory and have the 'iotc' prefix. To get started, navigate to the File tab and open iotc.h.

These documents are a complete reference for POSIX systems. Use the BSP to build the Device SDK on a custom, non-POSIX platform.
