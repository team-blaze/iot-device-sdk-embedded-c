Custom callback.

The Device SDK invokes callbacks to notify the client application after an operation completes.

enum iotc_crypto_key_signature_algorithm_e

Define a key signature algorithm to sign JWTs.

The client application must call create_iot_core_jwt() before iotc_connect(). Cite the same algorithm used to provision your key in Cloud IoT Core.

Note: RSASSA-PKCS1-v1_5 with SHA-256 (RS256) is not supported.

See also
create_iot_core_jwt

 - IOTC_CRYPTO_KEY_SIGNATURE_ALGORITHM_ES256 ECDSA with P-256 and
SHA-256.
